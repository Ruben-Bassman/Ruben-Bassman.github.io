JavaScript

  • Método: es una función que está asociada a un objeto. Los métodos se utilizan para realizar acciones sobre los datos almacenados en el objeto.
  • Función: es un bloque de código reutilizable que realiza una tarea específica y puede ser llamado desde otras partes del programa. Las funciones permiten organizar el código, evitar la duplicación y mejorar la legibilidad.
  • Propiedad: es una asociación entre una clave (nombre) y un valor dentro de un objeto. Las propiedades definen las características de un objeto y pueden contener cualquier tipo de dato, incluyendo otras funciones (métodos).
